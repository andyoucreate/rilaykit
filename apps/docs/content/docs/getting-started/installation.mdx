Rilay is modular. You can install only the packages you need. `@rilaykit/core` is always required.

## Basic Installation (Forms)

For simple forms, install the core and forms packages:

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab value="pnpm">
    ```bash
    pnpm add @rilaykit/core @rilaykit/forms
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    npm install @rilaykit/core @rilaykit/forms
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @rilaykit/core @rilaykit/forms
    ```
  </Tab>
</Tabs>

## Full Installation (With Workflows)

For multi-step workflows, add the workflow package (commercial license required):

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab value="pnpm">
    ```bash
    pnpm add @rilaykit/core @rilaykit/forms @rilaykit/workflow
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    npm install @rilaykit/core @rilaykit/forms @rilaykit/workflow
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @rilaykit/core @rilaykit/forms @rilaykit/workflow
    ```
  </Tab>
</Tabs>

## Package Overview

- **`@rilaykit/core`** (MIT): Core engine with configuration system, types, and validation
- **`@rilaykit/forms`** (MIT): Form builder for single-step forms
- **`@rilaykit/workflow`** (Commercial): Workflow engine for multi-step forms with advanced features

## Peer Dependencies

Rilay requires React 18+ and TypeScript 5+:

```json
{
  "peerDependencies": {
    "react": ">=18.0.0",
    "react-dom": ">=18.0.0",
    "typescript": ">=5.0.0"
  }
}
```
